# A股首板战法交易策略

## 策略概述
- **名称**：首板战法评分版
- **本金**：2000元
- **目标**：1个月内盈利
- **胜率目标**：80%+

## 核心逻辑
1. 筛选当日首板涨停股票（连板数=1）
2. 多维度评分（封板时间、价格、换手率、封板资金）
3. 选最高分股票买入
4. 次日冲高回落即卖出

## 评分规则（满分100）
| 维度 | 标准 | 加分 |
|------|------|------|
| 封板时间 | ≤10:00 | +20 |
| | ≤10:30 | +15 |
| | ≤11:00 | +10 |
| 价格 | ≤10元 | +15 |
| | ≤20元 | +10 |
| 换手率 | 3%-15% | +10 |
| 封板资金 | >1亿 | +10 |
| 基础分 | - | 50 |

## 交易规则
| 项目 | 设置 |
|------|------|
| 单票上限 | 1000元（50%仓位） |
| 止损线 | -3% |
| 止盈线 | +5% |
| 持仓时间 | 最多2天 |
| 每日最多 | 2只股票 |

## 每日流程
1. **09:25** 运行选股策略
2. **09:30** 开盘买入最高分股票
3. **14:55** 收盘前不涨停则卖出
4. **15:00** 记录交易日志

## 风险控制
- 严格止损，不补仓
- 单票不超过50%仓位
- 连续3天亏损暂停交易

## 代码文件
- `first_board_score.py` - 评分版策略
- `first_board_lite.py` - 简化版策略
- `backtest_first_board.py` - 回测验证

## 回测结果
- 历史胜率：59.4%
- 平均收益：+2.45%
- 优化方向：增加均线过滤、板块热点
